MultiGit - JGit example
========================

This example shows how to use multi\_git with jgit.

Requirements
--------------------

  - jruby > 1.7.0
  - bundler

Installation
--------------------

    $> bundle
    ... installs some gems ...
    $> jbundle
    ... installs some jars ...

You are done!

Usage
--------------------

    $> ruby test.rb
